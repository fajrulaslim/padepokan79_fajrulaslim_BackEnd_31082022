BAHASA PEMROGRAMAN YANG DIPAKAI:
1. PHP Versi 7 dengan bantuan Framework Codeigniter 3
2. Database MySQL

SOFTWARE YANG DIBUTUHKAN:
1. Apache HTTP Server: XAMPP / MAMP / Laragon / dsb...
2. MySQL database: Phpmyadmin / Mysql Workbench / Navicat / dsb ...
3. Browser: Google Chrome / Firefox / dsb ...
4. Code Editor (Optional): VS Code / Sublime Text / dsb ...

LANGKAH MENJALANKAN APLIKASI
1. Download dan extract file project, letakkan dalam folder dan beri nama "padepokan79".
2. Letakkan/pindahkan file project ke dalam folder Apache HTTP Server Anda.
3. Jalankan Apache HTTP Server Anda.
4. Di dalam folder project, terdapat file database dengan nama "padepokan79.sql". 
5. Pada software MySQL Database Anda, buat database dengan nama "padepokan79", kemudian import file database yang terdapat pada folder project tadi.
6. Buka browser, ketikkan alamat "localhost/padepokan79" pada kolom search bar.
7. Aplikasi siap digunakan.

CATATAN
1. Nama database harus sama dengan nama file database yang terdapat pada file project yaitu "padepokan79".
2. Jika ingin mengganti nama database ikuti langkah berikut:
	- Buka Software MySQL Database Anda, buat database dengan nama misal: "tes-fajrulaslim"
 	- Buka file "padepokan79.sql" dengan Code Editor, hapus baris 20 - 22
 	- Import file database
 	- Buka file "/application/config/database.php", pada baris 81 ganti nama database sesuai point pertama yaitu "tes-fajrulaslim"
